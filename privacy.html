Privacy Policy

This policy describes how the Pure Read browser extension (“the extension”) handles user data. Please read it before using the extension.

1. Data upload

The extension extracts and renders article content locally on your device. It does not upload article content, pages you read, or text you enter to any external server. Content extraction and reader-mode rendering are performed entirely in your browser.

2. Storage and sync

The extension uses Chrome’s chrome.storage (including chrome.storage.sync) to save your settings (e.g. theme, font size, reading width). This data is handled by Chrome’s sync mechanism; if you have Chrome sync enabled, it may sync across devices signed into the same Chrome account. We do not have access to or control over this stored data.

3. Network requests

The extension may fetch rule index and rule files (JSON) from a rules source you configure (e.g. your own server or a default rules repository) for content extraction and chapter navigation. These requests retrieve configuration data only; no remote code is executed. If you use chapter navigation or continuous reading, the extension may request next/previous chapter pages to extract content locally; such requests are made on behalf of the current page and the data is used locally, not sent to servers we operate.

4. Permissions

Permissions such as “access to all websites” are used to: inject reader mode on any webpage, request next/previous chapter pages when you use chapter navigation, and fetch rule data from your configured rules source. We do not use these permissions to collect your browsing history or identity for purposes unrelated to reading.

5. Changes and contact

We may update this privacy policy; the updated version will be posted on this page. If you have questions about this policy or how the extension handles data, you can contact us via the project repository (e.g. GitHub Issues) or the contact information in the documentation.

隐私政策

本政策说明 Pure Read 浏览器扩展（下称「本扩展」）如何对待用户数据。请在使用本扩展前阅读。

1. 数据是否上传

本扩展在您的设备上本地提取并渲染网页正文，不会将文章内容、您阅读的页面或您输入的文字上传到任何外部服务器。正文提取与阅读模式渲染均在浏览器内完成。

2. 存储与同步

本扩展使用 Chrome 提供的 chrome.storage（包括 chrome.storage.sync）保存您的设置（如主题、字体、阅读宽度等）。此类数据由 Chrome 的同步机制处理，若您开启 Chrome 账户同步，可能会在您登录的同一 Chrome 账户下的设备之间同步。我们无法访问或控制这些存储内容。

3. 与外部网络的交互

本扩展可能从您配置的规则/推广数据地址（如自建站点或默认规则仓库）拉取规则索引与规则详情（JSON 格式），用于正文提取与章节导航。此类请求仅获取配置数据，不执行远程代码。若您使用章节导航或连续阅读，扩展会请求下一章/上一章等页面内容以在本地提取正文，这些请求由扩展代表当前页面发起，数据在本地使用，不上传至我们控制的服务器。

4. 权限说明

本扩展声明的「访问所有网站」类权限用于：在任意网页上注入阅读模式、在您使用章节导航时请求下一章/上一章页面、以及从您配置的规则源地址拉取规则数据。我们不会利用这些权限收集您的浏览历史或身份信息用于与阅读无关的目的。

5. 变更与联系

我们可能会更新本隐私政策，更新后的版本将发布在本页面。若您对本政策或本扩展的数据处理有疑问，可通过项目仓库（如 GitHub）的 Issues 或文档中的联系方式与我们联系。
