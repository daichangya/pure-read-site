GitHub 主仓库 安装说明

Pure Read 扩展

Chrome 扩展（Manifest V3），在任意网页点击图标或使用快捷键即可进入阅读模式，正文在本地提取与渲染，不上传内容。

核心功能

一键阅读

扩展图标、快捷键 Ctrl+Shift+L（Mac：⌘+Shift+L）、右键菜单「使用阅读模式打开」。

排版与主题

浅色 / 深色 / 护眼等主题，字体与字号可调，预设风格（默认 / 紧凑 / 资讯 / 小说）。

阅读增强

阅读进度条、文章大纲、章节导航与连续阅读、搜索高亮、Focus 模式、TTS 朗读、高亮与笔记等。

导出与分享

导出 Markdown / HTML / 纯文本、打印与 PDF、截图、邮件与社交分享、高亮笔记导出。

配置持久化

设置保存在 chrome.storage.sync，可跨设备同步；白名单、黑名单、配置导入导出。

从源码安装（开发者模式）

  1. 1

    克隆 主仓库，执行 npm install、npm run build。

  2. 2

    打开 Chrome chrome://extensions/，开启「开发者模式」。

  3. 3

    「加载已解压的扩展程序」选择项目下的 dist 目录。

环境要求：Node.js 18+，Chrome 88+（支持 Manifest V3）。

Pure Read Rules

站点规则仓库，与主项目解耦，用于正文提取与章节导航。每条规则通过 host 与 CSS 选择器指定标题、正文、作者、排除区域及可选的下一章/上一章/目录。

谁会用

  • Pure Read 扩展：从自建网站或本地加载规则（MARKET_REMOTE_BASE_URL）。
  • 其他兼容本格式的阅读/抓取工具（见规则 schema 文档）。

规则分类

novel — 小说站 news — 新闻资讯 tech — 技术博客 general — 通用站点

贡献规则：Fork 仓库 → 在 rules/{category}/ 下新增或修改 JSON → 更新 index.json → 提交 PR。详见仓库内 CONTRIBUTING 与 docs。
